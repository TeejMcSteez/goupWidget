## GoUp QT Widget

Basic widget for my server monitor application, checkmark for ok X for services that are down and displays which services are down once clicked on.

## Settings/Configuration

Stored in system QT settings for Fedora 44 it is located at `~/.config/teejnet/GoUp.conf` but may vary depending on where QT app settings are stored 

## Dependencies

`sudo dnf install qt6-qtbase-devel`
